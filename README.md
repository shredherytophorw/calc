# Advanced Calculator

A Flask-based web calculator application with support for arithmetic expressions, history tracking, notes, currency conversion, measurement conversion, and settings.

## Features

- Basic calculator operations
- Safe mathematical expression evaluation
- Calculation history
- Notes section
- Currency conversion
- Measurement conversion
- Settings page
